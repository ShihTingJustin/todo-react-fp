# todo-react-fp
An Apple Reminders clone built with React.js.

## Setup
1. clone repository
```
git clone https://github.com/ShihTingJustin/todo-react-fp
```
2. go to project directory
```
cd todo-react-fp
```
3. install dependencies
```
yarn
```
4. serving the project
```
yarn dev
```
5. use it on
```
http://127.0.0.1:3000/
```

## Stack
* TypeScript
* React
* Redux
* Redux Toolkit
* Jest
* React Testing Library
* Tailwind
* Ant Design
